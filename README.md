# Data-AI-Specialist

# 🎓 Parcours de Certification : Data & AI Specialist

Bienvenue dans le niveau avancé ! Ce programme vous apprendra à transformer les données en décisions stratégiques et à déployer l'intelligence artificielle pour automatiser la croissance des entreprises.

---

## 🟠 Étape 1 : Expertise Financière et Audit (Bloc 1)
*Objectif : Devenir capable d'analyser la santé réelle d'une entreprise et d'utiliser la donnée comme preuve irréfutable.*

[**Accéder aux supports sur GitHub**](https://github.com/wanzo-academy/data-ai-specialist-b1)

| Module | Ce que vous allez apprendre | Ce que vous allez réaliser |
| :--- | :--- | :--- |
| **Score de Crédit** | Maîtriser les 5 piliers : Liquidité, Solvabilité, Stabilité, Activité et Intégrité. | Un rapport BI complet avec un plan d'action pour améliorer le score. |
| **Data Forensic** | Utiliser la Blockchain pour garantir que les données sont vraies et infalsifiables. | Un rapport d'audit technique (Forensic) sur les preuves financières. |
| **Stratégies de Croissance** | Simuler des flux de trésorerie pour maximiser la rentabilité. | Un plan d'optimisation avec des scénarios comparatifs. |
| **Reporting Avancé** | Créer des tableaux de bord pour suivre les marges nettes en temps réel. | Un dashboard interactif prêt à l'usage pour la direction. |
| **Analyse Prédictive** | Anticiper le futur financier grâce à des modèles de prévision. | Un système d'alertes automatiques pour les décisions critiques. |

**Total : 140 heures d'expertise**

---

## 🔵 Étape 2 : Intelligence Artificielle Agentique (Bloc 2)
*Objectif : Concevoir et déployer des agents IA capables d'automatiser les métiers et les processus.*

[**Accéder aux supports sur GitHub**](https://github.com/wanzo-academy/data-ai-specialist-b2)

| Module | Ce que vous allez apprendre | Ce que vous allez réaliser |
| :--- | :--- | :--- |
| **Agents IA (Wanzo Claw)** | Comprendre l'architecture des agents IA autonomes. | La documentation et l'architecture d'un agent métier. |
| **Contrôle de Gestion IA** | Automatiser le suivi du budget et l'analyse des écarts par l'IA. | Un workflow financier 100% automatisé avec reporting. |
| **Optimisation Commerciale** | Déployer des agents pour booster les ventes et le marketing. | Un plan d'action commercial automatisé par l'IA. |
| **Intégration No-Code** | Créer des agents complexes sans coder (support client, veille). | Des agents opérationnels configurés avec leur guide utilisateur. |

**Total : 120 heures d'innovation**
